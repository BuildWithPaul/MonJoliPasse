# Mon Joli Passe — API (Node.js / Express)

Service API basé sur Express. Il peut servir de backend léger (ex: endpoints utilitaires, proxy, stockage, etc.).  
Si votre usage est 100% front, ce composant peut être optionnel.

## Prérequis
- Node.js (LTS recommandé)
- Yarn (recommandé) ou npm

## Installation
```bash
yarn install
```

## Lancer en développement (rechargement auto)
```bash
yarn dev
```

## Lancer en production
```bash
yarn start
```

## Tests
```bash
yarn test
```

## Configuration
Si vous avez besoin de variables d’environnement, créez un fichier `.env` (non commité) et chargez‑le avec `dotenv`.
